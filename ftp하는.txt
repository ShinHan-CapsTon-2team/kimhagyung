Windows 운영체제에서 내장된 기능을 사용하여 간단한 FTP 서버를 활성화할 수 있습니다. 이는 주로 로컬 환경에서 테스트 및 파일 공유 목적으로 사용될 수 있습니다. 아래는 Windows 운영체제에서 내장된 FTP 서버를 활성화하는 방법을 설명하는 단계입니다:

1. Windows 기능 활성화/비활성화 열기:

2. Windows 검색 상자에 "Windows 기능 활성화 또는 비활성화"를 입력하고 해당 항목을 선택합니다.
Internet Information Services (IIS) 설치:

Windows 기능 창에서 "Internet Information Services" 옵션을 찾아 확장합니다.
"FTP Server" 옵션을 선택하고 하위 항목 중 "FTP 서비스"와 "FTP 서비스 확장 가능성"을 선택합니다.
변경 사항을 저장하기 위해 "확인"을 클릭합니다.
IIS Manager 실행:

시작 메뉴에서 "IIS Manager"를 검색하여 실행합니다.
FTP 사이트 생성:

IIS Manager에서 "컴퓨터 이름" > "사이트"를 선택한 다음, "사이트 추가" 옵션을 클릭합니다.
"사이트 추가" 대화상자에서 사이트의 이름을 입력하고, 물리적 경로로 원하는 폴더 경로를 선택합니다.
FTP 바인딩 및 인증 설정:

"FTP 바인딩 및 SSL 설정" 옵션을 선택하고, 포트 번호를 지정합니다. 기본적으로 21번 포트가 사용됩니다.
"SSL" 옵션을 설정하고자 한다면 SSL 인증서를 선택하고 설정합니다.
"인증" 옵션에서 사용자를 추가하고 암호를 설정합니다. 익명 사용자로 접속을 허용하거나, 사용자 인증을 요구할 수 있습니다.
FTP 서비스 시작:

IIS Manager에서 생성한 FTP 사이트를 선택한 다음, "시작"을 클릭하여 FTP 서비스를 시작합니다.
FTP 접속:

FTP 클라이언트 프로그램을 실행하고, Windows 컴퓨터의 IP 주소와 설정한 포트 번호로 접속합니다.
설정한 사용자 계정으로 로그인하여 파일을 업로드하거나 다운로드할 수 있습니다.
이렇게 하면 Windows 운영체제에서 내장된 기능을 사용하여 간단한 FTP 서버를 활성화하고 사용할 수 있습니다. 하지만 보안 상의 이유로 외부 접속을 허용하지 않거나, 안전하지 않은 환경에서 사용하는 것을 권장하지 않습니다. 필요한 경우 추가적인 보안 설정을 적용하고, SSL 인증서를 사용하여 암호화된 접속을 설정하는 것이 좋습니다.
